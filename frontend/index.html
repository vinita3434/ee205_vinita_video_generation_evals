HOW WE CALCULATE LATENCY & COST

Latency = time from when we send the request to the API until the video file is fully received (generation_end_time − request_start_time), in seconds.
Cost per model = we use fixed per-second rates from provider pricing: Kling 2.6 (fal.ai) and Sora 2 (OpenAI). Cost per video = video_duration_seconds × cost_per_second. These rates are set in backend config and can be updated when pricing changes.
